Trabalhando com Listas

Listas ordenadas

Uma lista ordenada é uma lista de itens apresentados em uma sequência, com cada item marcado de forma diferente para indicar a sua posição.
Aqui está um exemplo de lista ordenada, onde você ultiliza com a tag ol.

  1. Começo do dia = Acordar
  2. Tirar leite
  3. Faço um Duallingo
  4. Vou lá fora fazer o resto de serviço
  5. Me atualizo sobre o mercado financeiro, vejo e analiso opções de investimento
  6. Meio dia = Almoço

Listas não Ordenadas

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.
Aqui está um exemplo de lista não ordenada, onde você ultiliza com a tag ul.

  • Macarão
  • Arroz
  • Leite
  • Suco
  • Frutas
  • Pão
  • Café
  • Açucar
  • Sabão
  • Perfume
  • Shapoo
